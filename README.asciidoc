# Chumbok Starter Parent Example

## Check the pom.xml
```
<parent>
	<groupId>com.chumbok</groupId>
	<artifactId>chumbok-starter-parent</artifactId>
    <version>1.0.0</version>
</parent>
```

```
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
